🧠 MiniBrain MemoryService Architecture

Mission: Create a comprehensive memory system for MiniBrain to store, retrieve, and contextualize all chat conversations using Qdrant vector database for semantic search and long-term memory persistence.

1. System Overview

The MemoryService is designed to provide MiniBrain with persistent memory capabilities, enabling context-aware conversations and learning from historical interactions. The system leverages Qdrant's vector database for semantic similarity search and efficient retrieval of relevant conversation history.

4.1 Qdrant Service Implementation

public class QdrantMemoryService : IMemoryService { private readonly QdrantClient _qdrantClient; private readonly IEmbeddingService _embeddingService; private readonly ILogger<QdrantMemoryService> _logger; private const string CollectionName = "minibrain_conversations"; public async Task<string> StoreConversationAsync(ChatConversation conversation) { var embedding = await _embeddingService.GenerateEmbeddingAsync( $"{conversation.UserInput} {conversation.MiniBrainResponse}"); var point = new PointStruct { Id = conversation.Id, Vector = embedding, Payload = ConvertToPayload(conversation) }; await _qdrantClient.UpsertAsync(CollectionName, [point]); return conversation.Id; } }
